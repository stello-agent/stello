# 对话边界说明：批判性思维

## 本对话的目标
本子对话聚焦于“批判性思维”这一主题，作为父话题“什么是好的思考？”中的一个具体分支，讨论如何更清晰、更审慎、更有依据地分析观点、判断论证与形成结论。

## 可以讨论的内容
- 批判性思维的定义、特征与核心原则
- 批判性思维与“好的思考”之间的关系
- 如何识别论点、证据、前提、结论与推理链条
- 常见逻辑谬误、认知偏差与论证漏洞
- 如何评估信息来源的可信度与证据质量
- 如何提出澄清性问题、反例与替代解释
- 如何在日常学习、工作、阅读、写作和讨论中运用批判性思维
- 批判性思维与独立思考、理性讨论、决策质量之间的联系
- 提升批判性思维的方法、练习框架与具体步骤
- 对某个观点、论证或文本进行结构化分析，但重点应放在分析方法本身

## 不宜讨论的内容
- 与“批判性思维”无明显关联的泛泛人生哲学或广泛价值争论
- 仅表达立场、情绪或结论，而不涉及理由、证据或推理分析
- 偏离主题的知识介绍，例如单纯讲某一学科历史而不连接到思维分析
- 将本对话变成对具体社会、政治、伦理立场的辩论场，除非重点是分析论证方式
- 纯粹的说服、攻击、站队或情绪化评价
- 过度扩展到“所有思维方式”的完整体系讨论；如需讨论，应回到父话题或拆分新子话题
- 不以批判性思维为中心的实操建议，例如仅讨论时间管理、沟通技巧或学习习惯

## 讨论时的默认视角
- 优先关注“如何判断”而非“先相信什么”
- 优先分析论证质量，而非评价说话者身份
- 鼓励提出假设、限制条件与反例
- 区分事实、解释、观点与价值判断
- 允许保留不确定性，避免仓促下结论

## 与父话题的关系
本子对话回答的是：“好的思考”中的一个关键组成部分——人如何通过审查证据、检验推理和识别偏差，尽可能形成更可靠的判断。若讨论扩大到思维的整体品质、创造性、直觉、元认知等内容，应转回父话题或另建子对话。
