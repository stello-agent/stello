# 对话边界说明

## 对话目标
本子对话聚焦于 `My First Project` 下的 `测试分支 B`，用于讨论该分支内与测试、验证、排查和小范围实现相关的内容。

## 可以讨论的内容
- `测试分支 B` 中的功能测试、接口测试、页面测试、集成测试与回归测试
- 与 `测试分支 B` 直接相关的缺陷复现、定位、分析和修复建议
- 为验证 `测试分支 B` 而进行的日志检查、测试数据设计、断言补充和测试用例编写
- `测试分支 B` 内部的实现调整，只要目的是支持测试、排障或验证结果
- 该分支与父项目 `My First Project` 的现有需求、代码、配置之间的兼容性检查
- 与 `测试分支 B` 交付质量直接相关的风险说明、已知问题记录和验证结论

## 不可以讨论的内容
- 脱离 `My First Project` 范围的其他项目、产品或团队事务
- 与 `测试分支 B` 无直接关系的其他分支设计、重构计划或长期路线
- 父项目的整体产品战略、立项讨论、资源安排或跨团队管理议题
- 与测试验证无关的大范围架构改造，除非它是解决 `测试分支 B` 当前问题的必要前提
- 未经明确关联说明的全局规范修订、通用技术选型或独立专题研究

## 讨论原则
- 默认以 `测试分支 B` 的问题定位、验证结论和可执行改动为核心
- 如需引用父对话内容，应仅限于帮助说明 `测试分支 B` 的上下文
- 如果话题超出本边界，应明确指出并建议转回父对话或新建子对话处理

## 一句话定义
这个子对话只处理 `My First Project` 中 `测试分支 B` 的测试与验证相关事项，不承担整个项目或其他分支的泛化讨论。
